# Holert Consulting Website

Files:
- `index.html` = home page
- `biographies.html` = biographies page
- `styles.css` = all styling
- `script.js` = mobile menu
- `assets/` = put images here

To customize:
1. Edit text directly in `index.html` and `biographies.html`.
2. Replace contact info in both footers.
3. Add these optional image files:
   - `assets/home-hero.jpg`
   - `assets/bio-hero.jpg`
4. Open `index.html` in your browser to preview.

To deploy:
Upload all files to your hosting service/public folder. Keep the file names the same.
